Proximity

Disney+

disneyplus.com

The Proximity design principle describes elements of a web page as visual units. By placing related elements in proximity (=close) to eachother, structure is given to the page and its displayed information.

The Disney+ website shows this in how related elements are held close together, with big white spaces (in this case black) between unrelated groups of elements.

Alignment

Samsung

samsung.com

The Samsung webpage is an example for the Alignment design principle.

This principle states that every element on a page must have a visual connection with other elements, being either obvious ones or invisible lines. Elements should not be just randomly placed.

This is seen on this website, since related elements are vertically and/or horizontally aligned, e.g. in the grid with the products.

Contrast

Apple

apple.com

The website of Apple demonstrates the Contrast design principle very well:

It is used to strongly emphasise the difference between certain elements to display groups of elements. The alternating high contrast between background and font color works well to create visual interest, additionally it also helps improving the readability of the content.
